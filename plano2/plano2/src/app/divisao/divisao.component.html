Divisão

Divisão é uma operação matemática inversa da multiplicação. O ato de dividir por algum elemento de um conjunto só vaz sentido quando a multiplicação por aquele elemento for uma função bijetora (Onde ela retorna somente um resultado para cada operador) tomaremos como base apenas o conjunto dos números racionais representado pela letra "Q" caso não saiba sobre conjuntos númericos de uma olhada em Conjuntos Númericos. A vale ressaltar que a ideia de bijetividade não é satisfeita para o zero, sendo assim não se define divisão por zero. (caso deseje ver o conceito de divisão por zero de uma olhada aqui Divisão por zero.

Representação de uma operação de divisão

Essa é a forma usual de representar um divisão elementar. Ela pode ser encontrada também na representação de uma fração.

O primeiro termo ou no caso o número que será dividor chamamos de dividendo, o segundo termo ou o segundo número que será dividido chamamos de divisor. a propriedade fundamental da divisão diz que o divisor deve ser diferente 0. Pois qualquer divisão por 0 gera uma indeterminação.
